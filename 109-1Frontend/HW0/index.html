Windows 如何安裝Visual Studio Code

步驟一

進入Visual Studio Code 的網頁，可以上網打 VS code 也可以直接按我給的連結 => Visual Studio Code

進到以下的網站畫面後，點右上方的DOWNLOAD來下載

步驟二

接著會看到以下畫面，選windows版本的，之後電腦就會開始下載

步驟三

幾分鐘後會完成下載，接著執行.exe檔會看到以下畫面。點選我同意後再按下一步

步驟四

這邊選擇你要存檔的位置，預設會在C槽，而我自己改成D槽(如果沒有特殊需求可以不用改)，設定完後再按下一步

步驟五

這邊直接按下一步就好

步驟六

接下來會變成以下畫面，要將紅色勾勾的那兩行勾起來，勾好後再按下一步

步驟七

接下來直接按安裝，電腦就會開始做安裝了!!!

VS code 安裝幾個套件

按左邊工具列的第五個工具來做套件的查詢 *小提醒: 名字相同但大小寫不同就是不同的套件，要小心不要安裝錯

套件一: Chinese(Traditional)

功能:把介面從英文版改成中文版

下載之後要按右下角的 Restart Now 來從新啟動

套件二: Open In Default Browser直接開瀏覽器

功能:按ctrl 1會直接幫你打開瀏覽器

套件三: Beautify

功能:會自動幫你美化程式碼
